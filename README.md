## Abbreviations

Controls:

- J = Jump
- L = Light attack
- H = Heavy attack
- U = Use item (ex: arrow)
- M = Magic
- B = Block

Other:

- CC = Castle Crashers
- CCR = Castle Crashers Remastered
- OoB = Out of Bounds
